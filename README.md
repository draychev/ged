# ged is a Go (re)implementation of the venerable `ed` text editor


## Implemented Commands

 - `a` - append lines
 - `c` - not implemented (yet)
 - `d` - delete line(s)
 - `p` - print line(s)
 - `n` - print line(s) with line number
 - `w` - write file
 - `q` - quit
 - `i` - insert line
 - `t` - transfer (copy) line(s)
 - `m` - move line(s)
 - `x` - not implemented
 - `=` - show current line number
